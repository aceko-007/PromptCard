## 🚀 介绍

**PromptCard Desktop** 是一个为你精心设计的桌面应用程序，旨在帮助你安全、私密地管理和组织你的 AI 提示词（Prompts）。在这个 AI 驱动创作的时代，一个好的提示词就是一把金钥匙 🔑。本应用让你能够将这些宝贵的创意资产完全保留在自己的电脑上，无需担心云端同步的延迟或隐私泄露问题。

无论你是 AI 艺术家、作家、开发者还是营销人员，PromptCard Desktop 都能成为你强大的创意伙伴，让你的灵感永不枯竭！

## ✨ 核心功能

- **🗂️ 分类管理**: 通过直观的文件夹结构来组织你的提示词卡片。为不同的项目或主题创建独立的分类，让一切井井有条。

- **🏷️ 灵活的标签系统**: 为你的卡片添加多个模型标签（如 `Midjourney`, `Stable Diffusion`）和自定义标签，通过标签云快速筛选和定位你需要的灵感。

- **📸 一键卡片截图**: 轻松将任何卡片“快照”为图片。无论是为了分享你的绝妙提示词，还是为了在项目文档中归档，都非常方便。

- **⭐ 个人收藏夹**: 将你最常用或最喜欢的提示词标记为“收藏”，随时在专属的收藏视图中快速访问。

- **📋 快捷复制**: 一键复制卡片标题或完整的提示词内容到剪贴板，无缝对接到你的 AI 工具中。

- **🔗 外部链接集成**: 在卡片中附上相关的参考网站或社交媒体帖子链接，方便溯源和扩展阅读。

- **🔄 数据自由导入/导出**: 你的数据你做主！随时可以将整个卡片库导出为 `JSON` 文件进行备份，也可以轻松地从备份文件中恢复。


 


## 🛠️ 快速上手

1.  **下载并运行**: 前往 [下载](#-下载) 部分获取最新版本。解压后，直接运行 `PromptCard Desktop.exe` 即可启动。
2.  **创建你的第一张卡片**: 点击界面上的“添加卡片”按钮，开始填写你的第一个提示词。别忘了为它加上图片和标签！
3.  **组织与筛选**: 在左侧栏创建分类，在顶部的标签云中管理你的标签。点击标签即可快速筛选卡片。
4.  **探索更多**: 尝试截图、收藏、复制等功能，你会发现更多提高效率的妙用！
